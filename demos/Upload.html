    # Decode by Upload

    Finally, with QrcodeCapture comes with another component which allows image scanning via classic file upload. If you are on mobile and your browser supports it, your are not prompted with a file dialog but with your camera. So you can directly take the picture to be uploaded.

    Note, that nothing is actually uploaded. Everything is done client-side.

    # Source

    <template>
  <div>
    <p class="decode-result">Last result: <b>{{ result }}</b></p>

    <qrcode-capture @decode="onDecode" />
  </div>
</template>

<script>
export default {
  data () {
    return {
      result: ''
    }
  },

  methods: {
    onDecode (result) {
      this.result = result
    }
  }
}
</script>
